## Расположение библиотеки: 
  src/lib

## Описание
1. Библиотека реализует компоненты, для создания кастомного CLI приложения

2. Библиотека построена на базе библиотек commander
3. команды для библиотеки инициализируются через файлы конфингурации
4. Файлы конфигурации содержат параметры, необходимые для описания команды , ее аргументов и опций. И фунцию, которая получает параметры и опции команды и выполняет необходимое действие.
4. При запуске приложения, приложение ищет в определенной папке конфиги команд.
 - добавляет команды к приложению
 - и выполняет указанную пользователем при запуске команду
